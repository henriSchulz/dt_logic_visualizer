# dt_logic_visualizer

A tool for visualizing digital logic circuits and their symmetries.

## Features

- Interactive visualization of logic circuits
- Support for common logic gates (AND, OR, NOT, etc.)
- Highlighting of symmetrical structures
- Import/export circuit diagrams

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/dt_logic_visualizer.git
    ```
2. Install dependencies:
    ```bash
    cd dt_logic_visualizer
    npm install
    ```
3. Run the application:
    ```bash
    npm start
    ```

## Usage

- Use the editor to build and modify logic circuits.
- Click "Analyze Symmetry" to highlight symmetrical components.
